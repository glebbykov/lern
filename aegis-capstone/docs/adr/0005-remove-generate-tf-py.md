# ADR-0005: Удаление `generate_tf.py`, `*.tf` — единственный источник правды

## Status
Accepted (2026-04-29).

## Context
В `terraform/generate_tf.py` лежит Python-скрипт, который перезаписывает `variables.tf`, `azure.tf` и `inventory.tf` из захардкоженных строк.

На момент аудита `*.tf` файлы в репо **уже разошлись** с тем, что генерирует скрипт (ручные правки добавили мета-группы в inventory, ProxyCommand, фиксы peering-ресурсов). Любой запуск скрипта молча затрёт эти изменения без предупреждения.

Скрипт не приносит ценности: он не параметризован, не использует никакие данные кроме самих строк, не генерирует ничего динамического. Это дублирование с худшими свойствами.

## Decision
- `terraform/generate_tf.py` **удаляется**.
- Файлы `terraform/*.tf` — единственный источник правды для инфраструктурного кода.
- Если когда-нибудь возникнет реальная потребность в генерации — использовать `templatefile()`/`for_each` внутри Terraform, либо завести настоящий генератор с параметризацией и баннером `# DO NOT EDIT`.

## Consequences

### Положительные
- Ручные правки в `*.tf` больше не теряются.
- Один read-only путь обучения для нового контрибьютора: «правда — в `*.tf`».
- Уменьшается когнитивная нагрузка (один источник вместо двух конкурирующих).

### Отрицательные / Цена
- Если когда-либо понадобится массово размножать схожие ресурсы — придётся полагаться на `for_each` и `locals` (что и так уже сделано для VM/дисков).

### Что станет проще / сложнее в будущем
- **Проще:** code review — diff `*.tf` отражает реальные изменения, без необходимости проверять «а не сгенерируется ли это обратно».
- **Сложнее:** ничего.

## Alternatives considered

### Сохранить скрипт + добавить баннер `# DO NOT EDIT` в *.tf
Отвергли: пометка не блокирует ручные правки в IDE и не предотвращает молчаливое затирание. Кроме того, скрипт всё ещё не делает ничего полезного.

### Переписать скрипт на «настоящий» генератор с параметрами
Отвергли как преждевременную абстракцию. Сейчас 50 строк HCL, в которых есть смысл и от руки — генератор не нужен.
