# Урок 1: Процессор и Память

## Цель
Найти и обезвредить процесс, который утилизирует все ресурсы системы.

## Основные команды
- `uptime` — посмотреть среднюю загрузку (Load Average за 1, 5 и 15 минут).
- `free -h` — посмотреть свободную и занятую оперативную память.
- `top` / `htop` — диспетчер задач. Нажмите `P` для сортировки по CPU, `M` для сортировки по памяти.
- `ps aux --sort=-%cpu | head -n 10` — вывести топ-10 процессов по потреблению процессора.
- `pstree -p` — дерево процессов с их PID.
- `kill -9 <PID>` — принудительно завершить процесс.
- `killall <имя_процесса>` — убить все процессы с заданным именем.

## Задание
1. Запустите `./simulate.sh`.
2. Система начнет "тормозить".
3. Используя `top` или `ps`, найдите PID зловредного процесса (он маскируется под странным именем или `stress-ng`).
4. Убейте его с помощью `kill -9 <PID>` и убедитесь через `free -h` и `uptime`, что ресурсы освободились.
