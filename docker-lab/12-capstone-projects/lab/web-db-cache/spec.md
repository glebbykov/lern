# Capstone: Web + DB + Cache

## ТЗ
Соберите API сервис с Postgres и Redis, миграциями и healthchecks.

## Acceptance criteria
- `docker compose up -d` поднимает весь стек.
- API отвечает на `/healthz` и `/readyz`.
- Есть миграции схемы БД при старте.
- Добавлен мониторинг и ротация логов.

## Типовые баги для тренировки
- Не применяются миграции.
- Конфликт env-переменных.
- Потеря данных после перезапуска.
