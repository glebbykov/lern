# Level 5 — Функции, логирование, коды возврата

## Что нового на этом уровне

| Приём | Зачем |
|-------|-------|
| `function log()` | единый формат вывода с временной меткой |
| `function die()` | выход с сообщением об ошибке и кодом |
| `local` переменные | изоляция внутри функции, нет утечки в глобальный scope |
| `return <code>` | функция возвращает статус, вызывающий сам решает что делать |
| `$?` и `||` / `&&` | явная обработка кода возврата |

## Скрипты

- `find_large_files.sh` — поиск крупных файлов в директории с отчётом
- `file_report.sh` — статистика по файлам: количество, размер, расширения

## Ключевые идеи

```bash
# Функция с временной меткой
log() {
    echo "[$(date '+%Y-%m-%d %H:%M:%S')] $*"
}

# Завершение с кодом ошибки
die() {
    log "ERROR: $*" >&2
    exit 1
}

# Локальные переменные
count_files() {
    local dir="$1"
    local count
    count=$(ls -1 "$dir" 2>/dev/null | wc -l)
    echo "$count"
}

# Обработка кода возврата
result=$(count_files "$dir") || die "Не удалось подсчитать файлы"
```
