# Bash Scripts

Прогрессивное изучение Bash — от первой строки до production-скриптов.

## Уровни

| Уровень | Тема | Ключевые концепции |
|---------|------|--------------------|
| [level0](level0/) | Справочник | Проверки `-z`, `-d`, `-f`, `-r`, `-w`; one-liners |
| [level1](level1/) | Основы | Hardcoded логика, `ls`, `wc`, `echo` |
| [level2](level2/) | Аргументы | `$1`, `$2`, базовая параметризация |
| [level3](level3/) | Валидация | Проверка аргументов и директорий, `exit 1` |
| [level4](level4/) | Сложная логика | Фильтрация, glob, обработка нескольких источников |
| [level5](level5/) | Функции | `function`, `local`, `return`, `log()`, `die()` |
| [level6](level6/) | Расширенные паттерны | `getopts`, `trap`, lockfile, dry-run, конфиг-файл |
| [level7](level7/) | Production-ready | Полный шаблон: строгий режим, уровни лога, параллелизм |

## Прогрессия навыков

```
level1  →  "работает у меня"
level2  →  принимает аргументы
level3  →  не падает на плохих входных данных
level4  →  решает сложные задачи
level5  →  структурирован и читаем
level6  →  безопасен для автоматизации
level7  →  готов к продакшну
```
