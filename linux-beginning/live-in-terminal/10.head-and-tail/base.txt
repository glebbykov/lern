Описание:
head: по умолчанию выводит первые 10 строк файла, но можно указать другое количество строк с помощью параметра -n.

-n: выводит указанное количество строк.
-c: выводит указанное количество байт с начала файла.
-q: подавляет вывод заголовков, если просматривается несколько файлов.
tail: по умолчанию выводит последние 10 строк файла, но можно указать другое количество строк с помощью параметра -n.

-n: выводит указанное количество строк.
-c: выводит указанное количество байт с конца файла.
-f: следит за обновлением файла в реальном времени.
-q: подавляет вывод заголовков, если просматривается несколько файлов.
Расширенные примеры с параметрами
head:

Вывод первых 20 строк файла:
head -n 20 largefile.txt

Вывод первых 50 байт файла:
head -c 50 largefile.txt

Вывод первых 15 строк каждого из нескольких файлов без заголовков:
head -q -n 15 file1.txt file2.txt

tail:
tail -n 30 largefile.txt

Вывод последних 100 байт файла:
tail -c 100 largefile.txt

Постоянное наблюдение за обновлением файла в реальном времени:
tail -f largefile.txt

Вывод строк с конца файла, начиная с 5-й строки:
tail -n +5 largefile.txt
