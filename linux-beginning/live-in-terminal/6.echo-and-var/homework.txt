1. Создайте переменную color и присвойте ей значение вашего любимого цвета. Выведите сообщение с использованием этой переменной.
Пример вывода: My favorite color is [color].

2. Создайте переменную num_files, которая хранит количество файлов в текущей директории. Используйте командную подстановку и команду ls. Выведите количество файлов.
Пример вывода: There are [num_files] files in the current directory.

3. Создайте две переменные width и height, присвойте им числовые значения. Вычислите площадь прямоугольника и выведите результат.
Пример вывода: The area of the rectangle is [area].

4. Сохраните результат команды uname -r в переменную kernel_version. Выведите сообщение, содержащее текущую версию ядра Linux.
Пример вывода: The current kernel version is [kernel_version].

5. Создайте переменную logfile с именем файла для лога, например, system.log. Используйте команду echo для записи сообщения о запуске скрипта в этот файл.
Пример вывода: Script started. Log written to [logfile].

6. Создайте переменную path с полным путем до директории, в которой вы хотите создать новый файл. Создайте файл newfile.txt в указанной директории и выведите сообщение о его создании.
Пример вывода: File [path]/newfile.txt has been created.

7. Используйте переменные для хранения имени месяца и числа дней в этом месяце. Выведите сообщение, используя обе переменные.
Пример вывода: The month of [month] has [days] days.
