Базовая информация о командах mkdir и rmdir
mkdir: Команда mkdir используется для создания новых директорий.

Пример:

mkdir my_directory
Создаёт директорию с именем my_directory.
rmdir: Команда rmdir используется для удаления пустых директорий.

Пример:

rmdir my_directory
Удаляет директорию с именем my_directory, если она пуста.

Примеры использования
Создание новой директории:

mkdir new_folder
Создание нескольких директорий одновременно:


mkdir dir1 dir2 dir3
Удаление пустой директории:


rmdir dir1
Создание вложенных директорий с флагом -p:

mkdir -p parent/child/grandchild
