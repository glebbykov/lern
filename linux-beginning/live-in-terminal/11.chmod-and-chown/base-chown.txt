Команда chown
chown используется для изменения владельца и группы файла. Владелец файла может изменять права доступа к файлу, а группа может предоставлять совместный доступ нескольким пользователям.

Синтаксис:

chown [опции] владелец[:группа] файл
Основные параметры команды chown:
Изменение владельца файла: Используйте chown, чтобы назначить нового владельца для файла или директории.


chown username file.txt
Изменение владельца и группы файла: Для одновременного изменения владельца и группы используйте следующий формат:


chown username:groupname file.txt
Рекурсивное изменение владельца для всех файлов в директории: Параметр -R позволяет рекурсивно изменить владельца и группу для всех файлов в директории.


chown -R username:groupname /path/to/directory
Изменение только группы: Чтобы изменить только группу, используйте : перед именем группы, но без указания владельца.


chown :groupname file.txt
Примеры использования chown для стандартных путей Debian:
Изменить владельца файла /home/user1/data.txt на user2:


chown user2 /home/user1/data.txt
Изменить владельца и группу файла /etc/myconfig.conf на admin и группу config:


chown admin:config /etc/myconfig.conf
Рекурсивно изменить владельца всех файлов в директории /var/www/html на www-data:


chown -R www-data /var/www/html
Изменить только группу файла /var/log/auth.log на adm:


chown :adm /var/log/auth.log
Проверить текущего владельца файла: Чтобы проверить владельца и группу файла, используйте команду ls -l.


ls -l /etc/myconfig.conf
