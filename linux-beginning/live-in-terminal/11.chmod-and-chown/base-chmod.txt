Команда chmod
chmod используется для изменения прав доступа к файлам и директориям. В Linux существует три типа прав доступа: чтение (read, r), запись (write, w) и выполнение (execute, x). Эти права могут применяться к трем категориям пользователей:

владелец (user, u),
группа (group, g),
другие пользователи (others, o).
Синтаксис:

chmod [опции] права файл
Основные параметры команды chmod:
Изменение прав с использованием символьного представления:

u — права владельца (user),
g — права группы (group),
o — права других пользователей (others),
a — права всех категорий пользователей (all).
Вы можете добавлять или убирать права с помощью знаков + и -.

Пример:
Добавить право на выполнение для всех:


chmod a+x file.txt
Изменение прав с использованием числового представления (октального):

4 — чтение (read),
2 — запись (write),
1 — выполнение (execute).
Сочетание этих чисел даёт полные права. Например, число 755 означает:

7 (чтение, запись, выполнение) для владельца,
5 (чтение, выполнение) для группы,
5 (чтение, выполнение) для других.
Пример:


chmod 755 file.txt
Рекурсивное изменение прав для всех файлов в директории: Параметр -R позволяет рекурсивно изменить права для всех файлов в директории.


chmod -R 644 /path/to/directory
Примеры использования chmod для стандартных путей Debian:
Добавить права на выполнение для владельца файла /home/user/script.sh:


chmod u+x /home/user/script.sh
Установить права на чтение и запись для владельца и группы, а для других только чтение для файла /etc/myconfig.conf:


chmod 664 /etc/myconfig.conf
Удалить права на запись для других пользователей для файла /var/log/syslog:


chmod o-w /var/log/syslog
Установить права на чтение и выполнение для всех пользователей для файла /usr/local/bin/app:


chmod a+rx /usr/local/bin/app
Рекурсивно изменить права на все файлы в директории /var/www (чтение и выполнение для всех):


chmod -R 755 /var/www
