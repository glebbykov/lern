Объяснение пайпа (|) и команд ls и wc
Пайп (|) — это оператор в командной строке Unix/Linux, который позволяет передавать вывод одной команды в качестве ввода для другой команды. Это мощный способ комбинирования команд для последовательной обработки данных.

Команда ls: Используется для отображения списка файлов и директорий.
Пример:
ls
Этот пример выводит список всех файлов и директорий в текущей директории.

Опция -l для ls: Используется для отображения списка файлов в длинном формате, который включает подробную информацию о файлах, такую как права доступа, количество ссылок, владелец, размер, дата модификации и имя файла. Пример:
ls -l
Этот пример выводит список всех файлов и директорий с подробной информацией о каждом файле.

Команда wc: Используется для подсчета строк, слов и байтов в файлах или в потоке данных.
Пример использования с флагами:

wc -l: подсчитывает количество строк.
wc -w: подсчитывает количество слов.
wc -c: подсчитывает количество байтов (или символов).
Пример использования комбинации команд ls, wc и пайпа (|):
Подсчитайте количество файлов в директории:

ls | wc -l
В этом примере команда ls выводит список всех файлов и директорий, а wc -l подсчитывает количество строк в этом списке, что соответствует количеству файлов и директорий.
