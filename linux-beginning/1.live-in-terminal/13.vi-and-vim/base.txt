Редактор vi
Редактор vi — это один из самых популярных текстовых редакторов в Linux, доступный практически на всех системах. Он позволяет редактировать файлы в текстовом режиме.

Основные команды vi:
Открытие файла:

bash
Копировать код
vi имя_файла.txt
Это откроет файл в режиме команд.

Переход в режим вставки:

Нажмите i, чтобы войти в режим вставки и начать редактирование файла.
После этого вы можете вводить текст.
Сохранение изменений и выход:

Для сохранения изменений и выхода нажмите Esc, затем введите :wq и нажмите Enter.
Выход без сохранения:

Нажмите Esc, затем введите :q! и нажмите Enter, если хотите выйти без сохранения изменений.
Выход без сохранения (если изменений не было):

Нажмите Esc, затем введите :q и нажмите Enter, чтобы выйти из редактора, если не было внесено изменений.
Поиск по файлу:
Поиск строки текста вниз по файлу:

В командном режиме введите /, затем введите текст для поиска и нажмите Enter.
Для перехода к следующему вхождению нажмите n, для предыдущего — N.
Поиск строки текста вверх по файлу:

В командном режиме введите ?, затем введите текст для поиска и нажмите Enter.
Для перехода к предыдущему вхождению нажмите N, для следующего — n.
Дополнительные команды:
Удаление строки:
В командном режиме нажмите dd, чтобы удалить текущую строку.
Копирование и вставка:
В командном режиме нажмите yy, чтобы скопировать текущую строку.
Нажмите p, чтобы вставить строку.

Основные отличия vi и vim:
Vim (Vi Improved):

vim — это улучшенная версия редактора vi. Он добавляет множество функций, которые делают работу с текстом более удобной.
Некоторые из дополнительных функций:
Подсветка синтаксиса.
Автодополнение команд и слов.
Улучшенные возможности работы с буфером обмена (копирование и вставка текста).
Расширенные возможности поиска и замены текста.
Поддержка плагинов и макросов для расширения функциональности.
Vi:

Оригинальный редактор vi более простой и доступен на большинстве Unix-систем по умолчанию.
В нем отсутствуют многие дополнительные возможности, присутствующие в vim.
Если у вас есть возможность использовать vim, рекомендуется именно он для более удобной работы с текстом.

Замена слов в vi/vim:
Чтобы заменить слово или фразу в файле, можно использовать команду замены.

Замена всех вхождений слова в файле:
Войдите в командный режим, нажав Esc.
Введите следующую команду:

:%s/старое_слово/новое_слово/g
Это заменит все вхождения слова старое_слово на новое_слово в файле.

Замена одного вхождения слова:
Если вы хотите заменить только одно вхождение слова:

Перейдите на строку, где находится это слово.
Введите команду:

:s/старое_слово/новое_слово/g

Замена с подтверждением:
Чтобы заменить с подтверждением на каждом шаге, добавьте c в конце команды:

:%s/старое_слово/новое_слово/gc
При каждой замене система будет запрашивать подтверждение (нажмите y, чтобы подтвердить замену, или n, чтобы пропустить).

Вызов нумерации строк:
Чтобы включить отображение номеров строк в vi или vim, используйте команду:

Войдите в командный режим, нажав Esc.
Введите следующую команду:

:set number
Это включит нумерацию строк. Если хотите отключить нумерацию, используйте команду:

:set nonumber
Эти команды помогут более эффективно работать с текстом в редакторах vi и vim, включая замену слов и просмотр нумерации строк.
