Введение в команды head, tail и less
Команды head, tail, и less являются базовыми инструментами для работы с текстовыми файлами в Linux. Они используются для просмотра содержимого файлов, что важно при анализе данных, отладке системных логов и работе с большими файлами.

1. Команда head
Команда head отображает первые строки файла. По умолчанию она выводит первые 10 строк, но это можно изменить с помощью опций.

Основные опции head:

-n <число> — указывает количество строк, которые нужно вывести.
Пример:
head -n 5 filename.txt
Выводит первые 5 строк файла.
2. Команда tail
Команда tail выводит последние строки файла. По умолчанию выводятся последние 10 строк.

Основные опции tail:

-n <число> — указывает количество строк, которые нужно вывести.

-f — отображает последние строки в реальном времени, что полезно для просмотра логов.

Пример:

tail -n 3 filename.txt
Выводит последние 3 строки файла.

Для мониторинга файлов (например, логов) в реальном времени:

tail -f /var/log/syslog
3. Команда less
Команда less позволяет построчно просматривать содержимое больших файлов. Она удобна, так как не загружает весь файл сразу, а работает по мере необходимости, позволяя быстро и удобно просматривать большие файлы.

Основные команды в less:

Space — для перехода к следующей странице.
b — для перехода к предыдущей странице.
q — для выхода из просмотра.
Пример:
less filename.txt
