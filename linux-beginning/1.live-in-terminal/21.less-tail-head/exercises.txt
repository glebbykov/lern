Самостоятельная работа
Цель: Освоить базовые команды для просмотра и анализа файлов (head, tail, и less), научиться их эффективно использовать для работы с большими файлами.

Задачи:

Используйте команду head для просмотра первых 5 строк файла.
С помощью команды tail выведите последние 5 строк системного файла /var/log/syslog.
Примените команду less для построчного просмотра большого файла.
Примените команду tail -f для мониторинга изменения системного журнала в реальном времени.
Шаги:

Создайте файл с несколькими строками текста и примените команду head для вывода первых 5 строк.
Используйте tail -f для отслеживания изменений в логах или другом динамическом файле.
Откройте большой файл с помощью less и попробуйте перемещаться по нему с помощью клавиш Space, b, и q.
