Домашнее задание
1. Часть 1: Работа с cut
Создайте файл с содержимым:

Копировать код
John,Doe,30,Engineer
Jane,Smith,28,Designer
Jack,Johnson,35,Developer
Используя команду cut, извлеките первые два поля (имена и фамилии) из каждого ряда. Используйте запятую как разделитель.

Извлеките только третье поле (возраст) из каждой строки и выведите его отдельно.

2. Часть 2: Работа с paste
Создайте два текстовых файла. Один с именами:

Копировать код
John
Jane
Jack
Второй с фамилиями:

Копировать код
Doe
Smith
Johnson
Используйте команду paste, чтобы объединить имена и фамилии в одну строку, используя пробел в качестве разделителя.

Попробуйте объединить те же строки, используя запятую или другой символ-разделитель.

3. Часть 3: Комбинация команд cut и paste
Возьмите файл с данными (например, список пользователей с их ролями) и с помощью cut извлеките только имена.

Используйте команду paste, чтобы объединить эти имена с другой информацией (например, с номерами телефонов или email-адресами, хранящимися в другом файле).

Изучите, как можно комбинировать команды cut и paste для автоматической обработки больших наборов данных.
