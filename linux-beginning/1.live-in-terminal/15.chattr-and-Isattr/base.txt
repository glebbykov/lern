Создание пользователей и групп

Основная теория:
В Linux управление пользователями и группами играет ключевую роль в контроле доступа к ресурсам системы. Каждый пользователь принадлежит хотя бы одной группе, и права доступа могут быть заданы как для отдельных пользователей, так и для групп.

Пользователь — это сущность, которая может войти в систему и выполнять действия. Пользователи могут быть системными (для сервисов и демонов) и реальными (человеческими).
Группа — это коллекция пользователей, которая позволяет управлять правами доступа к файлам и директориям для нескольких пользователей одновременно.
Основные команды для работы с пользователями и группами:
Создание нового пользователя:

sudo useradd имя_пользователя
Эта команда создает нового пользователя в системе. После создания рекомендуется задать пароль:


sudo passwd имя_пользователя
Создание новой группы:


sudo groupadd имя_группы
Эта команда создает новую группу.

Добавление пользователя в группу: Чтобы добавить существующего пользователя в группу, используется команда:


sudo usermod -aG имя_группы имя_пользователя
Просмотр информации о пользователях и группах:

Чтобы увидеть, в какие группы входит пользователь:

groups имя_пользователя
Просмотр списка всех пользователей:

cat /etc/passwd
Просмотр списка всех групп:

cat /etc/group
Удаление пользователя: Чтобы удалить пользователя, используйте команду:

sudo userdel имя_пользователя
Удаление группы: Для удаления группы:

sudo groupdel имя_группы
