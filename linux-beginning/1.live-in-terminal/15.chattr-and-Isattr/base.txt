Команды chattr и lsattr в Linux
Основное применение
chattr (change attributes) — команда, которая изменяет атрибуты файлов в файловых системах Linux, таких как ext4. 
Это позволяет задавать дополнительные ограничения на файл, делая его неизменяемым, защищённым от удаления, или предотвращая запись в файл.

lsattr (list attributes) — команда, которая отображает текущие атрибуты файлов или директорий. 
Это полезно для проверки того, какие атрибуты установлены на файле.

Основные атрибуты для chattr
i — неизменяемый файл. Когда атрибут установлен, файл не может быть изменён, удалён или переименован.

Пример использования:

sudo chattr +i myfile.txt
Устанавливает атрибут неизменяемости на файл myfile.txt. Попытки его изменения или удаления приведут к ошибке.
a — разрешает добавление данных в файл, но запрещает удаление или изменение существующих данных.

Пример использования:

sudo chattr +a logfile.txt
Теперь можно только добавлять данные в конец файла logfile.txt.

Основные опции для lsattr
lsattr просто перечисляет атрибуты файлов в формате:

lsattr myfile.txt
Вывод будет отображать, какие атрибуты установлены на данном файле.

Использование с опцией -R для рекурсивного просмотра атрибутов файлов в директориях:
lsattr -R /path/to/directory

Примеры:
Установим атрибут неизменяемости для файла:
sudo chattr +i important.conf

Проверим атрибуты файла:
lsattr important.conf

Снимем атрибут:
sudo chattr -i important.conf
