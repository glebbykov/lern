Базовая информация о командах cat и more
cat: Команда cat используется для отображения содержимого одного или нескольких файлов. Она также может использоваться для создания файлов и объединения нескольких файлов в один.

Пример:
cat file1.txt

Выводит содержимое файла file1.txt на экран.
more: Команда more используется для постраничного просмотра содержимого файлов. Она удобна для просмотра больших файлов, так как позволяет пролистывать их по одной странице.

more file1.txt
Показывает содержимое файла file1.txt постранично.
Примеры использования
Просмотр содержимого файла с помощью cat:

cat notes.txt

Просмотр содержимого нескольких файлов с помощью cat:
cat file1.txt file2.txt
Постраничный просмотр содержимого файла с помощью more:

more largefile.txt
Создание файла с использованием cat и перенаправлением:

cat > newfile.txt
Затем вводите текст, чтобы создать файл. Нажмите Ctrl+D для завершения ввода.
