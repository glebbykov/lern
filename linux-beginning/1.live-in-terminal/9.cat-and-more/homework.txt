1.Создайте два файла chapter1.txt и chapter2.txt. В каждый файл добавьте по 3 строки текста.
2.Объедините содержимое файлов chapter1.txt и chapter2.txt в новый файл book.txt.
3.Добавьте содержимое файла chapter2.txt в конец файла chapter1.txt.
4.Просмотрите содержимое файла book.txt с помощью команды cat.
5.Создайте новый файл notes.txt с помощью команды cat > notes.txt и введите текстовые заметки (не менее 5 строк).
6.Создайте файл large_notes.txt, который будет содержать не менее 50 строк текста. Вы можете сгенерировать его с помощью команды seq

seq 1 300 | sed 's/^/Note /' > large_notes.txt

7.Используйте команду more для просмотра содержимого файла large_notes.txt постранично.
8.Создайте файл log.txt и добавьте в него несколько записей лога с помощью команды cat >> log.txt. (можно использовать glebbykov/lern/linux-beginning/live-in-terminal/4.grep/system.log)
9.Создайте файл final_report.txt, содержащий результат объединения трех файлов log.txt, large_notes.txt, и book.txt, а также добавьте в этот файл сообщение о дате создания с использованием команды date.
