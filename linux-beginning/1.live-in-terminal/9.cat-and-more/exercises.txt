1. Создайте файл story.txt с помощью команды cat и запишите в него несколько строк текста.
2. Выведите содержимое файла story.txt на экран с использованием команды cat.
3. Создайте два файла part1.txt и part2.txt, затем объедините их содержимое в файл full_story.txt с использованием команды cat.
4. Используйте команду more для постраничного просмотра содержимого файла full_story.txt.
