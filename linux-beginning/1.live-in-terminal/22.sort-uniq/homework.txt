Домашнее задание
Часть 1: Работа с sort

Создайте файл с несколькими строками данных, включая числа и текст.
Используйте sort -n для сортировки чисел и sort -r для сортировки строк в обратном порядке.
Часть 2: Работа с uniq

Создайте файл с дублирующимися строками.
Используйте uniq -c для подсчёта числа вхождений каждой строки.
Примените uniq -d, чтобы вывести только повторяющиеся строки.
Часть 3: Комбинирование команд

Создайте большой файл с перемешанными строками.
Используйте комбинацию sort | uniq, чтобы удалить все дубликаты и отсортировать строки.
Дополнительно:

Исследуйте команды sort и uniq для работы с файлами, содержащими числа, и выясните, как использовать их для сортировки чисел в различных форматах (например, десятичных).
