Самостоятельная работа
Цель: Освоить команды для сортировки и удаления дубликатов строк с помощью sort и uniq.

Задачи:

Используйте команду sort для сортировки строк файла по алфавиту и по числовым значениям.
Примените команду uniq для удаления подряд идущих дубликатов строк.
Используйте комбинацию sort и uniq, чтобы отсортировать файл и удалить все дубликаты строк.
Шаги:

Создайте файл с повторяющимися строками и отсортируйте его с помощью sort.
Примените команду uniq для удаления дубликатов и подсчёта числа вхождений каждой строки.
Исследуйте, как команда sort -u работает без использования uniq.
