
Введение в команды uniq и sort
Команды uniq и sort в Linux часто используются для работы с текстовыми данными, особенно при обработке списков и сортировке строк. Эти команды полезны для упорядочивания данных, удаления дубликатов и анализа текстовой информации.

1. Команда sort
Команда sort используется для сортировки строк в файле или выводе команды. Она может сортировать строки по алфавиту, числовым значениям и другим критериям.

Основные опции sort:

-r — сортировка в обратном порядке (по убыванию).

-n — числовая сортировка.

-u — удаление дубликатов во время сортировки.

Пример:

sort filename.txt
Это выведет строки файла filename.txt в алфавитном порядке.

Для числовой сортировки:

sort -n filename.txt
2. Команда uniq
Команда uniq удаляет повторяющиеся строки из отсортированного файла или вывода команды. Она работает только с повторяющимися строками, которые идут подряд. Поэтому часто используется вместе с sort.

Основные опции uniq:

-c — подсчёт количества вхождений каждой строки.

-d — выводит только дубликаты (строки, которые повторяются).

-u — выводит только уникальные строки.

Пример:

uniq filename.txt
Это удалит подряд идущие дубликаты строк в файле.

Как использовать sort и uniq вместе
Так как uniq удаляет только подряд идущие дубликаты, перед использованием этой команды нужно отсортировать файл с помощью sort.

Пример удаления всех дубликатов:

sort filename.txt | uniq
Если хотите также подсчитать, сколько раз каждая строка встречалась в файле:

sort filename.txt | uniq -c
