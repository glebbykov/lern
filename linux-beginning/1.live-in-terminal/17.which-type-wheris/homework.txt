Домашнее задание
1. Часть 1: Работа с командой which
Найдите полный путь к следующим программам:
cat
awk
python

Используйте опцию -a, чтобы проверить, есть ли в системе несколько версий этих программ.

Определите, какие программы находятся ближе всего в вашем $PATH. Для этого измените переменную $PATH, добавив в неё новый каталог, и проверьте, как изменится результат команды which.

2. Часть 2: Анализ команд с type
Определите, является ли команда alias встроенной в оболочку или внешней программой.

Используйте опцию -a команды type, чтобы увидеть все варианты, как команда ls может интерпретироваться в системе. Попробуйте определить, есть ли алиасы или функции, связанные с этой командой.

3. Часть 3: Использование whereis
С помощью команды whereis найдите страницы справки, исполняемые файлы и исходные коды для программ make, find, и ssh.

Попробуйте найти исходный код для команд, скомпилированных из исходников. Если исходники недоступны через whereis, выясните, где они могут быть размещены в системе или интернете.

4. Дополнительно: Настройка и анализ $PATH
Изучите значение переменной $PATH на вашей системе:

echo $PATH
Добавьте новый каталог в $PATH, создайте скрипт внутри этого каталога, и убедитесь, что он доступен для выполнения через which.

Удалите созданный каталог из $PATH и проверьте, как это влияет на доступность команды, которую вы добавили.
