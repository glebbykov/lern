Команда which
Команда which в Linux используется для поиска местоположения исполняемого файла программы. Она показывает путь к файлу, который будет выполнен, если вы введете имя команды в терминале. Это полезно для понимания, какой именно бинарный файл будет использован, если одна и та же программа установлена в нескольких местах.

Основной синтаксис:

which <команда>
Пример использования:

which python
Вывод:

/usr/bin/python
Этот вывод показывает, что когда вы вводите python, система выполнит бинарный файл, находящийся по пути /usr/bin/python.

Команда type
type — команда, которая показывает информацию о том, что собой представляет команда: это встроенная команда оболочки (shell built-in), алиас, функция или внешний исполняемый файл.

Основной синтаксис:

type <команда>
Пример использования:

type ls
Вывод:

ls is aliased to `ls --color=auto`
Этот вывод показывает, что ls — это алиас на команду ls --color=auto.

Команда whereis
whereis — команда для поиска исполняемых файлов, исходного кода и страниц man (руководств) для определенной программы. В отличие от which, которая ищет только исполняемый файл, whereis может также найти дополнительные ресурсы, связанные с командой.

Основной синтаксис:

whereis <команда>
Пример использования:

whereis gcc
Вывод:

gcc: /usr/bin/gcc /usr/lib/gcc /usr/share/man/man1/gcc.1.gz
Этот вывод показывает путь к исполняемому файлу gcc, библиотекам и странице справки (man).
