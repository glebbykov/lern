Самостоятельная работа
Цель: Изучить команды which, type, и whereis для поиска исполняемых файлов, анализа встроенных команд и работы с документацией в Linux.

Задачи:
Использовать команду which для поиска местоположения различных программ и анализировать все возможные варианты их размещения с помощью опции -a.
Применить команду type для определения, являются ли команды встроенными в оболочку, алиасами или внешними программами.
Использовать команду whereis для поиска исполняемых файлов, исходных кодов и страниц справки программ.

Шаги:
Используйте команду which для поиска путей к программам bash, python, и nano. Проверьте, есть ли несколько версий этих программ в системе с помощью опции -a.
С помощью команды type определите, является ли команда cd встроенной командой оболочки или внешней программой. Повторите это для команд echo, grep, и ls.
Примените команду whereis для поиска исполняемых файлов и документации для таких программ, как git, curl, и gcc. Используйте опции -b, -m и -s, чтобы изучить разные аспекты этих программ.
