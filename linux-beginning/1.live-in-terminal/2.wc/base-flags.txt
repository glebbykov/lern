wc - word count (количество слов)

l: Подсчитывает количество строк.
Пример использования:
wc -l file.txt
Выводит количество строк в файле file.txt.

-w: Подсчитывает количество слов.
Пример использования:
wc -w file.txt
Выводит количество слов в файле file.txt.

-c: Подсчитывает количество байтов.
wc -c file.txt
Выводит количество байтов (символов) в файле file.txt.

-m: Подсчитывает количество символов.
wc -m file.txt
Выводит количество символов в файле file.txt. Это полезно для подсчета символов в файлах с многобайтовыми символами, где количество байтов может не соответствовать количеству символов.

-L: Показывает длину самой длинной строки в символах.
wc -L file.txt
Выводит длину самой длинной строки в файле file.txt.

Примеры комбинирования флагов и указания нескольких файлов:
Вы можете использовать несколько флагов одновременно для получения разных типов информации сразу. Например:
wc -l -w -c file.txt
Этот пример выведет количество строк, слов и байтов для файла file.txt.

wc -l -w -c homework.txt additional.txt
Этот пример выведет количество строк, слов и байтов для файлов homework.txt и additional.txt.

По итогу:
wc -l — количество строк
wc -w — количество слов
wc -c — количество байтов
wc -m — количество символов
wc -L — длина самой длинной строки
