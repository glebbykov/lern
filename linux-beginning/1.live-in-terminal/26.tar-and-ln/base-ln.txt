
Команда `ln` в Linux

Команда `ln` используется для создания ссылок на файлы или директории в Linux. Ссылки позволяют обращаться к файлам или каталогам под разными именами или с разных мест в файловой системе. Существует два типа ссылок: жёсткие и символические.

Типы ссылок:

1. Жёсткая ссылка (Hard link): Жёсткая ссылка является дубликатом файла. Она указывает на те же данные на диске, что и оригинальный файл. При удалении одного из файлов (ссылки или оригинала) данные сохраняются, пока существует хотя бы одна ссылка на них.

2. Символическая ссылка (Symbolic link): Символическая ссылка указывает на путь к другому файлу. Если оригинальный файл удалён, символическая ссылка станет недействительной ("битой").

Основные аргументы команды `ln`:

1. `-s` — создание символической ссылки вместо жёсткой. Используется для указания на файл по его пути.
   Пример:
 
   ln -s /путь/к/файлу ссылка
 

2. `-f` — принудительное создание ссылки, перезаписывая существующий файл или ссылку с тем же именем.
   Пример:
 
   ln -sf /путь/к/файлу ссылка
 

3. `-v` — вывод подробной информации о создании ссылки.
   Пример:
 
   ln -sv /путь/к/файлу ссылка
 

4. `-i` — запрос подтверждения при перезаписи существующих файлов.
   Пример:
 
   ln -si /путь/к/файлу ссылка
 

5. `-T` — рассматривает цель как файл и создаёт ссылку, даже если цель существует как каталог.
   Пример:
 
   ln -sT /путь/к/файлу ссылка
 

Примеры использования:

- Создание жёсткой ссылки:

  ln original.txt hardlink.txt


- Создание символической ссылки:

  ln -s original.txt symlink.txt


- Перезапись существующей ссылки:

  ln -sf original.txt symlink.txt


Команда `ln` является мощным инструментом для управления файлами и директориями, позволяя создавать эффективные связи между разными частями файловой системы.
