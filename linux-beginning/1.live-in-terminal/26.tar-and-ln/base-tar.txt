Команда tar в Linux
Команда tar (от "tape archive") используется для создания, распаковки и управления архивами в Linux. С помощью tar можно объединить несколько файлов и директорий в один архивный файл (обычно с расширением .tar, .tar.gz, .tgz и т.д.) или распаковать их обратно. tar особенно полезен для создания резервных копий и передачи большого количества файлов.

Основные аргументы команды tar:
-c — создать архив (create).

Используется для создания нового архива.

Пример:
tar -cvf archive.tar /path/to/directory
-x — извлечь архив (extract).

Используется для распаковки архива.

Пример:
tar -xvf archive.tar
-f — указать имя архива (file).

Указывает имя архива для создания или распаковки. Этот флаг всегда используется вместе с именем файла архива.

Пример:
tar -cvf my_archive.tar file1 file2
-v — показать подробный вывод (verbose).

Позволяет видеть, какие файлы добавляются или извлекаются. Полезно для отслеживания процесса создания или распаковки архива.

Пример:
tar -cvf archive.tar file1 file2
-z — сжатие с помощью gzip.

Используется для сжатия архива с помощью утилиты gzip, создавая файл с расширением .tar.gz или .tgz.

Пример:
tar -cvzf archive.tar.gz /path/to/directory
-j — сжатие с помощью bzip2.

Сжимает архив с помощью bzip2, создавая файл с расширением .tar.bz2.

Пример:
tar -cvjf archive.tar.bz2 /path/to/directory
-J — сжатие с помощью xz.

Используется для создания архива, сжатого с помощью xz, с расширением .tar.xz.

Пример:
tar -cvJf archive.tar.xz /path/to/directory
-t — просмотр содержимого архива без распаковки.

Полезно для проверки содержимого архива.
Пример:

tar -tvf archive.tar
-C — указание директории для распаковки.

Позволяет указать, в какую директорию распаковать архив.
Пример:

tar -xvf archive.tar -C /path/to/extract

Примеры использования:
Создание архива из директории:

tar -cvf backup.tar /home/user/documents
Создание сжатого архива с gzip:

tar -cvzf backup.tar.gz /home/user/documents
Распаковка архива:

tar -xvf backup.tar
Распаковка сжатого архива .tar.gz в указанную директорию:

tar -xvzf backup.tar.gz -C /home/user/extracted_files
Просмотр содержимого архива:

tar -tvf backup.tar
