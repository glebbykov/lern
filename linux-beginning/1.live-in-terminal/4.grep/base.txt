Общее описание grep
grep — это утилита командной строки в Unix/Linux, которая используется для поиска определённых строк или шаблонов в файлах или выводе команд. 
Она принимает в качестве аргумента регулярное выражение и ищет строки, соответствующие этому шаблону.

Основные флаги для grep:
-i: Игнорирует регистр букв при поиске.

Пример: 
grep -i "pattern" file.txt
найдёт все вхождения pattern, независимо от регистра.

-v: Инвертирует поиск, выводя строки, которые не соответствуют шаблону.
Пример: 
grep -v "pattern" file.txt 
выведет все строки, которые не содержат pattern.

-r или -R: Рекурсивный поиск, который ищет шаблон во всех файлах и поддиректориях.
Пример: 
grep -r "pattern" /path/to/directory 
ищет pattern во всех файлах внутри директории.

-l: Выводит только имена файлов, содержащих совпадения, а не сами строки.
Пример: 
grep -l "pattern" *.txt 
выведет имена всех .txt файлов, содержащих pattern.

-c: Подсчитывает количество строк, соответствующих шаблону, и выводит это число.
Пример: 
grep -c "pattern" file.txt 
выведет количество строк, содержащих pattern.

-n: Выводит номера строк, в которых найдены совпадения.
Пример: 
grep -n "pattern" file.txt 
покажет строки с совпадением и их номера.

-e: Позволяет указать несколько шаблонов поиска.
Пример: 
grep -e "pattern1" -e "pattern2" file.txt 
ищет pattern1 и pattern2.

-w: Ищет только полные слова, соответствующие шаблону.
Пример: 
grep -w "word" file.txt 
ищет только слово word, а не его части.

-A: Показывает строки после совпадения (A - After).
Пример: 
grep -A 2 "pattern" file.txt
выводит совпадающие строки и 2 строки после них.

-B: Показывает строки перед совпадением (B - Before).
Пример: 
grep -B 2 "pattern" file.txt 
выводит совпадающие строки и 2 строки до них.

-С: Показывает строки перед и полсле совпадения.
Пример: 
grep -С 2 "pattern" file.txt 
выводит совпадающие строки и 2 строки до них и 2 после них.
