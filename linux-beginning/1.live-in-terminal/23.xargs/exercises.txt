1. Использование `xargs` для создания директорий

Создайте несколько директорий, имена которых указаны в файле dirs.txt. Используйте `xargs` для автоматизации создания директорий.
Пример:
cat dirs.txt | xargs mkdir

2. Ограничение количества одновременно выполняемых команд

Удалите файлы из файла file_list.txt, но удаляйте их по 3 файла за один вызов команды.
Пример:
cat file_list.txt | xargs -n 3 rm

3. Передача данных с фильтрацией

Используйте команду find для нахождения всех файлов с расширением .log в текущей директории и удалите их с помощью `xargs`.
Пример:
find . -name "*.log" | xargs rm

4. Проверка длины аргументов

Используйте xargs с опцией -s, чтобы передавать команды, содержащие не более 100 символов.
Пример:
cat file_list.txt | xargs -s 100 rm

5. Обработка пустых строк

Проверьте, как `xargs` работает с пустыми строками, используя опцию -r.
Пример:
cat file_list.txt | xargs -r rm
