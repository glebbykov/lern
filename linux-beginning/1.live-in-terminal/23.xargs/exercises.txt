1.Создайте несколько директорий, имена которых указаны в файле dirs.txt. Используйте `xargs` для автоматизации создания директорий.
2.Удалите файлы из файла file_list.txt, но удаляйте их по 3 файла за один вызов команды.
3.Используйте команду find для нахождения всех файлов с расширением .log в текущей директории и удалите их с помощью `xargs`.
4.Используйте xargs с опцией -s, чтобы передавать команды, содержащие не более 100 символов.
5.Проверьте, как `xargs` работает с пустыми строками, используя опцию -r.
