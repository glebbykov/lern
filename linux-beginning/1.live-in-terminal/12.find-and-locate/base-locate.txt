Команда locate
locate — это команда для быстрого поиска файлов в системе. В отличие от find, она использует предварительно сгенерированную базу данных файловой системы, 
что делает её поиск быстрее, но не всегда актуальным, так как база обновляется с некоторой задержкой.

Утилита не установлена по умолчанию поэтому установить её можно комнадой:
  sudo apt install locate

Синтаксис:
locate [путь или шаблон_файла]

Основные параметры команды locate:
Простой поиск файла:
locate filename.txt
Эта команда находит все файлы с именем filename.txt, зарегистрированные в базе данных.

Поиск по части имени файла: Вы можете искать файлы по части имени, используя символы подстановки.
locate "*.log"
Ограничение количества результатов: Параметр -n позволяет ограничить количество найденных файлов.
locate --limit=10 "*.log"
Обновление базы данных: Команда locate использует базу данных, которая периодически обновляется с помощью утилиты updatedb. Чтобы обновить базу вручную, выполните:
sudo updatedb

Примеры использования locate:
Найти все файлы, содержащие слово "config" в имени:
locate config

Найти все файлы с расширением .conf и ограничить результат 5 файлами:
locate -n 5 "*.conf"

Обновить базу данных и выполнить поиск файла system.log:
sudo updatedb
locate system.log
