Команда find
find — это мощная команда для поиска файлов и директорий по различным критериям, таким как имя файла, дата последнего изменения, размер, тип файла и многое другое.

Синтаксис:
find [начальная_директория] [условия_поиска] [действия_с_результатом]
Основные параметры команды find:
Поиск по имени: Используйте параметр -name для поиска файлов или директорий по имени.

find /path/to/search -name "filename.txt"
Эта команда ищет файл с именем filename.txt в указанной директории /path/to/search и всех её поддиректориях.

Поиск по части имени (с использованием шаблонов): Вы можете использовать символы подстановки (*) для поиска файлов, соответствующих шаблону.

find /path/to/search -name "*.log"
Этот пример находит все файлы с расширением .log в указанной директории.

Поиск по дате изменения: Используйте параметр -mtime для поиска файлов, изменённых за последние N дней.

find /path/to/search -mtime -7
Эта команда ищет все файлы, изменённые за последние 7 дней.

Поиск по размеру: Параметр -size позволяет находить файлы, размер которых больше или меньше указанного значения.
+100M — файлы больше 100 мегабайт.
-50k — файлы меньше 50 килобайт.

find /path/to/search -size +100M

Поиск по типу файла: Параметр -type позволяет искать только файлы (f), директории (d), символические ссылки (l) и т.д.
find /path/to/search -type f
Этот пример находит только файлы в указанной директории.

Выполнение действия с найденными файлами: Команда find позволяет не только искать файлы, но и выполнять с ними определённые действия, например, удаление.
find /path/to/search -name "*.tmp" -delete
Эта команда удаляет все временные файлы (*.tmp) в указанной директории.

Поиск с учётом прав доступа: Вы можете искать файлы по их разрешениям с помощью параметра -perm.
find /path/to/search -perm 644
Этот пример находит все файлы с правами доступа 644.

Поиск файлов, принадлежащих конкретному пользователю: Используйте параметр -user, чтобы искать файлы, принадлежащие определённому пользователю.

find /path/to/search -user username
Поиск с использованием нескольких условий: Вы можете комбинировать условия поиска с помощью параметров -and, -or, и ! (не).

find /path/to/search -name "*.txt" -or -name "*.log"
Этот пример ищет все файлы с расширениями .txt и .log.

Примеры использования find:
Найти все файлы с расширением .sh в домашней директории:

find ~/ -name "*.sh"
Найти и удалить все файлы старше 30 дней:

find /path/to/search -mtime +30 -delete
Найти все файлы размером больше 500 мегабайт:

find /path/to/search -size +500M
Найти все директории, принадлежащие пользователю user1:

find /path/to/search -type d -user user1
Найти файлы с правами доступа 755 и скопировать их в другую директорию:

find /path/to/search -perm 755 -exec cp {} /backup/directory \;
