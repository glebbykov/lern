Создание файлов с последующей настройкой прав:

Задача: Создайте три файла в своей домашней директории: file1.txt, file2.txt, file3.txt. После этого:
Установите для всех файлов права на чтение и запись только для владельца.
Установите право на выполнение для всех пользователей только для файла file3.txt.

Задача: Создайте скрипт test.sh в своей домашней директории. Для этого скрипта:
Установите права на чтение, запись и выполнение для владельца.
Установите права на чтение и выполнение для группы и других пользователей.
Выполните это с помощью символьного представления, а затем числового.

Задача: Создайте директорию mydir в своей домашней директории, а также 2 подкаталога внутри нее. В каждом подкаталоге создайте по 2 файла. Далее:
Установите права на чтение и выполнение для всех пользователей для всех файлов и каталогов внутри mydir.
Используйте рекурсивную команду для изменения прав.
