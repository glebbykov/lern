Настройка прав для системных файлов:

Задача: Создайте два файла в директории /home/yourusername/system_configs — config1.conf и config2.conf. Затем:
Для файла config1.conf установите права на чтение для всех пользователей.
Для файла config2.conf установите права на чтение и запись только для владельца.

Рекурсивное изменение прав на веб-контент:

Задача: Создайте директорию web_project в своей домашней директории. Внутри этой директории создайте несколько подкаталогов, например css, js, images, и несколько файлов в каждом подкаталоге. После этого:
Установите права на выполнение для всех подкаталогов.
Установите права на чтение для всех файлов в этих подкаталогах.

Настройка прав для скриптов:

Задача: Создайте директорию scripts в своей домашней директории. Внутри директории создайте несколько скриптов с расширением .sh. Затем:
Установите право на выполнение для владельца и группы для всех этих скриптов.
Для одного из скриптов (на выбор) установите права на чтение и выполнение для всех пользователей.
