Самостоятельная работа: Команды `tr` и `tee`

1. Используйте команду `tr` для замены всех пробелов на подчеркивания в тексте, находящемся в файле text.txt.
Пример:
```bash
cat text.txt | tr ' ' '_'
```

2. Примените команду `tr` для перевода всех букв в верхний регистр.
Пример:
```bash
cat text.txt | tr '[:lower:]' '[:upper:]'
```

3. Используйте команду `tr`, чтобы удалить все цифры из текста.
Пример:
```bash
cat text.txt | tr -d '0-9'
```

4. Используйте команду `tee` для записи вывода команды `ls` в файл `file_list.txt` и одновременно выводите результат на экран.
Пример:
```bash
ls | tee file_list.txt
```

5. Примените команду `tee` с опцией `-a` для добавления строки "Добавлено" в конец файла `file_list.txt`.
Пример:
```bash
echo "Добавлено" | tee -a file_list.txt
