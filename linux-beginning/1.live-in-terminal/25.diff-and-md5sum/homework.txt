Домашнее задание: команды diff и md5sum
Сравнение файлов с большими различиями:

Создайте два файла с большим количеством различий (например, с 10-20 строками текста).
Используйте команду diff -c для вывода различий в контекстном формате.
Использование команды md5sum для проверки нескольких файлов:

Сохраните контрольные суммы для нескольких файлов в один файл (например, all_checksums.md5).
Измените один из файлов.
Используйте команду md5sum -c, чтобы проверить все файлы сразу на целостность.
Создание и проверка контрольной суммы для бинарного файла:

Возьмите любой бинарный файл (например, скопируйте утилиту из bin) и вычислите его контрольную сумму с помощью опции -b.
Измените файл, добавив или удалив некоторые данные (например, измените имя файла).
Проверьте контрольную сумму для файла до и после изменения.
