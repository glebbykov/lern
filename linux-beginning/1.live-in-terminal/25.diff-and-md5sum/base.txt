Основное применение команд diff и md5sum
Команда diff:
Команда diff используется для сравнения содержимого двух файлов и отображения различий между ними. 
Она полезна для проверки изменений в текстовых файлах и нахождения строк, которые были добавлены, изменены или удалены.

Основные атрибуты команды diff:

-u — выводит различия в виде унифицированного формата (удобно для патчей и версий файлов).
-c — выводит различия в контекстном формате (с несколькими строками до и после изменений).
-i — игнорирует различия в регистре букв (например, "Hello" и "hello" будут считаться одинаковыми).
-w — игнорирует пробелы и табуляции при сравнении файлов.

Пример использования:
diff file1.txt file2.txt

Команда md5sum:
Команда md5sum используется для вычисления и проверки контрольной суммы (хеш-суммы) файла с помощью алгоритма MD5. Она помогает убедиться в том, что файл не был изменён или повреждён.

Основные атрибуты команды md5sum:

-c — проверяет контрольные суммы файлов на основе ранее сохранённого списка.
-t — вычисляет хеш-сумму только для текста (внутри ASCII-символов).
-b — вычисляет хеш-сумму для бинарных данных.
Пример использования:

Команда выводит контрольную сумму файла, которую можно использовать для проверки целостности файла после копирования или скачивания.
