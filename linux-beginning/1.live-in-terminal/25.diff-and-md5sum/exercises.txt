Самостоятельная работа: команды diff и md5sum
Сравнение файлов с помощью diff:

Создайте два текстовых файла file1.txt и file2.txt.
Добавьте в каждый файл различный текст.
Используя команду diff, найдите различия между этими файлами.
Использование diff -u для удобного сравнения:

Используйте команду diff с опцией -u, чтобы вывести различия между файлами в унифицированном формате.
Создание контрольных сумм с помощью md5sum:

Создайте два текстовых файла и вычислите их контрольные суммы с помощью команды md5sum.
Проверка целостности файла с помощью контрольных сумм:

Сохраните контрольную сумму одного из файлов в отдельный файл (например, checksum.md5).
Измените содержимое исходного файла.
Используйте команду md5sum -c для проверки целостности файла на основе сохранённой контрольной суммы.
